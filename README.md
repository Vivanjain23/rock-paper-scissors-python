# 🎮 Rock Paper Scissors Game (Python)

A simple command-line Rock Paper Scissors game built using Python where a player competes against the computer.

## 📌 Features
- Player vs Computer gameplay
- Random computer choice using Python random module
- Multiple rounds
- Score tracking
- Final winner announcement

## 🛠 Technologies Used
- Python
- Random Module

## ▶️ How to Run
1. Download the repository
2. Run the Python file:

python rock_paper_scissors_game.py

## 📚 Learning Outcome
This project helped me practice:
- Loops
- Conditional statements
- User input handling
- Basic game logic
